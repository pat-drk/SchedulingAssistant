Teams Shifts Assistant

A localized scheduling engine designed for high-precision staff coordination and seamless Microsoft Teams integration.

The Teams Shifts Scheduling Assistant is a specialized browser application for managing staff assignments across multiple time segments. By utilizing local SQLite technology, your data remains secure and private on your workstation.

6. Core Features

Daily Run Board

The main scheduling interface. Coordinates date, segment, and role-based assignments with real-time conflict checking.

Assignment Process

  1. 1. Select Date and Segment.
  2. 2. Locate Role Card.
  3. 3. Click [+ Add] for availability-filtered list.
  4. 4. Confirm selection.

Auto-Fill Engine

System-generated suggestions based on training, availability patterns, and history weights.

8. Workflow Examples

Daily Routine

  1. 01 Open your .db file.
  2. 02 Select Date in Daily Run.
  3. 03 Fill gaps manually or via Auto-Fill.
  4. 04 Use Ctrl+S to commit changes.

  1. 01 Set Month in Monthly tab.
  2. 02 Copy from previous month.
  3. 03 Review Weekday Overrides.
  4. 04 Apply Defaults to board.

10. Troubleshooting

"No database loaded"

Click New DB or Open DB in the top bar to activate the system.

Changes not saving

Verify write permissions for your .db file and ensure no other user has it locked.
